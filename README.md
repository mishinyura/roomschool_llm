# roomschool_llm

1. Перед началом работы требуется установить Docker через команду ```bash sh get-docker.sh```, 
предварительно убедившись, что файл есть в корне проекта, в ином случае скачать его 
через команду ```bash curl -fsSL https://get.docker.com -o get-docker.sh```
2. Выдай права на исполнение файлу инициализации ssl и запусти скрипт командой ```bash chmod +x init-ssl.sh ./init-ssl.sh```.
После выполнения команды скрипт запрост подтверждение.
